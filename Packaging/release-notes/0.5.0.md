ContainerDesktop 0.5.0

中文
相较于 0.3.1，0.5.0 是一次面向日常使用效率和资源管理深度的功能更新。0.3.1 主要补齐了软件更新说明的多语言展示；0.5.0 在此基础上扩展了全局快速操作、资源关联、文件浏览编辑、系统清理、Docker 兼容终端和发布更新链路，让 Container Desktop 更适合作为长期打开的 container 工作台。

新增全局快速操作和更丰富的导航入口。用户可以更快跳转到页面、资源详情、任务历史和常用动作；容器、镜像、卷、网络和 Compose 项目也会提供更贴近上下文的快捷命令。侧边栏、Dashboard 和资源列表的入口也做了整理，常用信息更容易被发现。

资源管理能力进一步增强。容器端口现在会生成更实用的打开、复制和访问建议；容器、镜像、卷、网络和 Compose 之间的关联信息会在详情页中集中呈现。文件浏览增加目录信息面板，并支持对容器、Machine 和 Volume 中的文本文件进行更清晰的预览与编辑。

系统维护和运行体验也有明显改进。System 页面新增独立清理面板，便于安全清理已停止容器和 dangling 镜像；应用支持开机启动偏好；外部终端选择、Docker 兼容终端系统集成和 Finder 服务入口进一步完善。日志和终端显示增加快照回放与覆盖输出压缩，减少长时间命令输出带来的噪音。

Compose、观测和操作历史的细节也得到加强。Compose 项目操作、服务观察面板、日志预设、任务历史抽屉和操作提示都更完整；更新检查继续兼容多语言 appcast，并保留旧字段兼容性。此版本还补充了覆盖快速操作、资源关联、文件编辑、开机启动、系统清理、终端滚动和更新解析等场景的测试。

English
Compared with 0.3.1, version 0.5.0 is a feature release focused on daily workflow speed and deeper resource management. Version 0.3.1 mainly completed multilingual release notes for software updates. Version 0.5.0 builds on that foundation with global quick actions, resource associations, file browsing and editing, system cleanup, Docker Compatibility Terminal improvements, and a stronger release/update pipeline.

Global quick actions are now broader and more useful. Users can jump to pages, resource details, task history, and common commands faster. Containers, images, volumes, networks, and Compose projects now expose more contextual actions, while the sidebar, Dashboard, and resource lists have been refined to make frequent workflows easier to discover.

Resource management is deeper. Container ports now produce practical open, copy, and access suggestions. Related containers, images, volumes, networks, and Compose projects are grouped in detail pages. File browsing adds folder information panels, with clearer preview and editing support for text files inside containers, Machines, and volumes.

System maintenance and runtime ergonomics have also improved. The System page now includes a dedicated cleanup panel for safely removing stopped containers and dangling images. The app adds launch-at-login preferences, improves external terminal selection, strengthens Docker Compatibility Terminal system integration, and expands Finder service support. Logs and terminal output now use snapshot replay and overwrite compaction to reduce noise from long-running commands.

Compose, observability, and operation history receive a set of focused refinements as well. Compose project actions, service observation, log presets, task history, and operation toasts are more complete. Update checks continue to support multilingual appcasts while preserving compatibility with legacy fields. This release also adds tests for quick actions, resource associations, file editing, launch at login, cleanup, terminal scrolling, and update parsing.
