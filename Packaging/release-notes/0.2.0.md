ContainerDesktop 0.2.0

中文
本次更新重点提升 Compose 稳定性、系统环境可见性和资源管理体验。System 页面新增组件版本展示与手动检查更新能力，可查看 container、container-compose 及运行时组件版本；Compose 修复重启后历史任务误显示运行中、启动失败原因不可见的问题，并修正 Apple container 默认 builder/vminit latest 镜像失效导致 build 类 Compose 无法启动的情况。界面上优化了设置窗口左右分区滚动和全局细滚动条，日志、抽屉、详情页等滚动区域更清爽。资源管理方面增强了镜像详情、Machine 文件浏览、Stats 历史监控、注册表浏览和观测能力，并补充了更多错误诊断与操作提示，整体使用和排障更顺畅。

English
This release focuses on Compose reliability, clearer system environment visibility, and a smoother resource management experience. The System page now shows component versions and supports manual update checks for container, container-compose, and runtime components. Compose now repairs stale running task history after restart, surfaces startup failure details directly, and fixes failed build-based Compose startup caused by unavailable Apple container default builder/vminit latest images. The UI also improves split scrolling in Settings and uses thinner global scroll bars, making logs, drawers, and detail pages cleaner. Resource management is expanded with richer image details, Machine file browsing, Stats history monitoring, registry browsing, observability improvements, and clearer diagnostics and operation guidance.
