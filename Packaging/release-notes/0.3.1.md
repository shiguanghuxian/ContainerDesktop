ContainerDesktop 0.3.1

中文
相较于 0.3.0，这个版本不是资源管理或 Docker 兼容终端的功能扩展，而是一次针对软件更新链路的补齐。0.3.0 已经带来了 Docker 兼容终端、镜像仓库筛选、网络详情页、System 组件版本折叠展示，以及 appcast 发布信息写入；0.3.1 在这些能力保持不变的基础上，让发布说明能按语言正确展示。

软件更新现在使用独立的 `release_notes_en` 和 `release_notes_zh_hans` 字段。About 页会根据应用语言显示对应内容；选择“跟随系统”时使用解析后的系统语言，中文系统显示简体中文，其他语言默认显示英文。

兼容性也做了补强：appcast 继续写入旧版 `release_notes` 字段，内容与英文说明保持一致，旧客户端仍可看到更新说明；新版客户端也能读取旧 `release_notes` / `body` 字段，并尽量拆分历史“中文 / English”分段 Markdown，避免从 0.3.0 或更早版本升级时出现空白说明。

English
Compared with 0.3.0, this release is not a new expansion of resource management or the Docker Compatibility Terminal. It is a focused follow-up to complete the software update pipeline. Version 0.3.0 introduced the Docker Compatibility Terminal, image repository filtering, Network detail pages, compact System component version display, and appcast release metadata; version 0.3.1 keeps those capabilities unchanged and makes update notes display in the right language.

Software updates now use separate `release_notes_en` and `release_notes_zh_hans` fields. The About page displays the matching notes for the app language setting. When the app follows the system language, it uses the resolved system language, shows Simplified Chinese for Chinese systems, and defaults to English for other languages.

Compatibility is preserved as well: appcasts still write the legacy `release_notes` field with the same English content, so older clients continue to show release information. Newer clients can also read older `release_notes` / `body` fields and split existing `中文 / English` Markdown sections where possible, preventing blank notes when upgrading from 0.3.0 or earlier appcasts.
